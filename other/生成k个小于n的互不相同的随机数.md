《编程珠玑》习题1.4：

 **如何生成位于0至n - 1之间的k个不同的随机顺序的随机整数**？尽量使你的程序简短高效。 

```c
void swap(int *a, int *b) 
{  
    int tmp = *a;  
    *a = *b;  
    *b = tmp;  
}  
void produce (int a[], int n, int k) 
{  
    int i;  
    //对数组a依次赋一个不同的值  
    for (i = 0; i < n + 1; i++) 
    {  
        a[i] = i + 1;  
    }  
    srand((int)time(0));  
    //下面的语句用于产生n个不同的随机数，存于数组的0到n-1位中  
    // i + rand() % (n - i)产生一个范围i到n的随机数  
    //那么将这个下标的数组数据和以i为下标的数组数据swap肯定不重复  
    for (i = 0; i < k; i++)
    {
        swap(&a[i], &a[i+rand()%(n-i)]);
    } 
}  
```
